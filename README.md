# Security-Log-Analyzer
This project simulates a lightweight SIEM-style log analysis tool. It parses authentication logs, detects suspicious behavior, and generates a list of anomalies that a cybersecurity analyst would escalate.

# What it detects
  1.Brute-force login attempts
  2. Successful login after multiple failed attempts
  3. Suspicious external IPs
  4. Off-hours login activity
  5. Multiple account access from on IP

# Purpose
project demonstrates incident response skills such as log parsing, anomaly detection, and building detection rules
